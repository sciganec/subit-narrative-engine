## A Dobre Phrasebook (DBR Edition)

*Common expressions, greetings, wisdom sayings, and ceremonial phrases in the resonant language of Dobre ‚Äî using only **D, B, R** consonants and **a, e, i, o** vowels.*

---

## üéØ **How to Use This Phrasebook**

Each entry includes:
- **Dobre** ‚Äî the spoken phrase (using only D, B, R)
- **Archetypes** ‚Äî the original SUBIT archetypes
- **Meaning** ‚Äî how to use it in conversation
- **Context** ‚Äî when to say it

**Pronunciation tips:**
- Speak slowly and clearly
- Let each syllable resonate
- Pause slightly between words (indicated by spaces)
- All words follow the pattern: `[D/B/R] + vowel`

**Grammar Notes:**
- `re` = "is/are/exists" (question word, from AUTUMN)
- `ba` = yes (from NORTH)
- `bo` = no (from SOUTH)
- `de` = you (YOU)
- `di` = I/me (ME)
- `do` = we (WE)
- `da` = they (THEY)
- `ro` = now (SUMMER)
- `ra` = before/why (WINTER)
- `ri` = later/what (SPRING)
- `be` = where (WHERE)
