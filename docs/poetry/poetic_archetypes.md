#### 00 00 00 ‚Äî ZERO
**THEY-NORTH-WINTER**

| Dimension | Value |
|-----------|-------|
| **Poetic Voice** | Silence. The unwritten poem. The white space before the first word. |
| **Imagery** | Blank page, untouched snow, empty mirror, void, breath withheld |
| **Meter** | No meter ‚Äî the pause between heartbeats |
| **Rhyme** | No rhyme ‚Äî the potential for all rhyme |
| **Forms** | Silence, blank page, unwritten book |
| **Key Words** | nothing, zero, void, silence, before, unborn, unwritten |

**Example Couplet:**
```
Before the first word was spoken,
before the first line was broken ‚Äî
nothing.
```

2. Read its poetic voice description
3. Use its imagery cluster for metaphors
4. Follow its metrical tendencies
5. Write from its point of view

### For Analyzing Poetry
1. Identify the poem's dominant voice (WHO)
2. Note its primary imagery (WHERE)
3. Feel its temporal rhythm (WHEN)
4. Match to the closest archetype
5. Understand the poem's archetypal foundation
